## Customer Shopping Behavior Analysis 🛍️

###### 📌 Project Overview

This project provides a comprehensive analysis of customer shopping patterns to drive data-driven business decisions. By leveraging SQL for data extraction, Python for deep-dive exploratory data analysis (EDA), and Power BI for interactive visualization, this project identifies key customer segments and purchasing trends.

###### 🛠️ Tech Stack

Data Analysis: Python (Pandas, Matplotlib, Seaborn)

Database Management: MySQL / PostgreSQL

Visualization: Power BI \& Microsoft PowerPoint

Environment: Jupyter Notebook / VS Code

###### 📂 Project Structure

customer\_behavior\_sql\_queries.sql: Advanced SQL scripts for data cleaning and KPI calculation.

Customer\_Shopping\_Behavior\_Analysis.ipynb: Python notebook containing statistical analysis and data distribution plots.

customer\_behavior\_dashboard.pbix: Interactive Power BI dashboard for executive reporting.

Customer-Shopping-Behavior-Analysis.pptx: Final presentation of business recommendations.

###### 🚀 Key Insights \& Features

Customer Segmentation: Identified high-value customers based on purchase frequency and average order value (AOV).

Trend Analysis: Seasonal shopping trends were analyzed to optimize inventory management.

Payment Preferences: Breakdown of preferred payment methods to streamline the checkout process.

Category Performance: Visualized top-performing product categories by revenue and volume.
